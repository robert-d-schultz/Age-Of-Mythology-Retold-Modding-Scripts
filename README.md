CryBarEditor can handle .tmm/.tma, I would use that instead of these scripts. Find it here: [https://github.com/CryShana/CryBarEditor/releases/tag/1.1.1](https://github.com/CryShana/CryBarEditor)

The .py files run in Blender's scripting environment. Adjust the path to in the script and run it.
Select a model before running .tmm exporter.
You can select an existing model/armature before you run the .tma script and it will put the animation there.

The .bt files are 010 editor templates. They are probably irrelevent now since the file format specs are released within the game's modding documentation.
